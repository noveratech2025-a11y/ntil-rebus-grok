# NTIL REBUS-GROK: Complete Deployable System

This is the fully integrated, production-ready codebase. Every file is complete and functional.

## Quick Start

### Prerequisites
- Java 21+
- Node.js 20+
- Docker & Docker Compose
- Maven 3.8+
- Git

### Development Setup

```bash
make setup
make dev
```

### Build Production

```bash
make build
```

### Run Tests

```bash
make test
```

## Core Components

### Backend (Spring Boot 3.2, Java 21)
- REST API for communication analysis
- Grok API integration for AI analysis
- Swarm intelligence engine (500 agents)
- PostgreSQL database
- Redis caching

### Frontend (React 18)
- Real-time dashboard
- Communication analysis interface
- Advisory report viewer
- Risk assessment visualization
